# AI Resume Analyzer (Flask)

Simple demo Flask app to upload a resume (PDF or text), extract basic skills, and compare them against common skills for a chosen job role.

## Features

- Upload resume as PDF or text
- Extract skills via simple keyword-based NLP
- Choose a target job role (Software Engineer, Data Scientist, Frontend Developer)
- View matched skills, missing skills, and other detected skills

## Setup

1. Create and activate a virtual environment (recommended).
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
python app.py

```
